# This is NOT the Next.js you know

This project was migrated from Expo/React Native to Next.js (App Router) on
2026-07-14 to get reliable Web Push notifications without an Apple Developer
account. See CONTEXT.md for the full history and reasoning.

Next.js has breaking changes vs your training data — APIs, conventions, and
file structure may all differ. Read `node_modules/next/dist/docs/` before
writing any code. Heed deprecation notices.
